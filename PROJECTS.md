## 프로젝트 2: ERP 발주 자동화 (자비스 AI)

### 개요
거래처가 SMS/카카오톡으로 발주 → AI가 분석 → 경영박사 ERP에 자동 입력

### 왜 필요한가
- 하루 30건 발주를 사람이 일일이 읽고 ERP에 입력 중
- 거래처마다 제각각으로 주문 ("닭", "생닭", "9호", "후라이드용" → 모두 같은 것)
- 입력 오류, 시간 낭비 발생

### 목표 시스템 흐름
```
거래처 문자/카톡
    ↓
SMS 자동 수집 (Tasker/MacroDroid)
    ↓
AI 주문 분석 (OpenAI API)
    ↓
품목코드 변환 (D001, D002...)
    ↓
주문 검수 화면 (담당자 확인)
    ↓
경영박사 ERP 자동 입력 (pywinauto)
    ↓
엑셀 저장 (pandas)
    ↓
완료 알림
```

### 핵심 데이터 구조 (설계 확정)

**거래처 테이블**
| 거래처ID | 거래처명 | 전화번호 |
|----------|----------|----------|
| C001 | OO치킨 | 010-xxxx |
| C002 | XX치킨 | 010-xxxx |

**품목 테이블**
| ERP코드 | 품목명 |
|---------|--------|
| D001 | 생닭 9호 |
| D002 | 순살 |
| S001 | 양념소스 |

**거래처별 별칭 테이블 (핵심!)**
| 거래처 | 표현 | ERP코드 |
|--------|------|---------|
| OO치킨 | 닭 | D001 |
| OO치킨 | 생닭 | D001 |
| XX치킨 | 큰닭 | D001 |
| XX치킨 | 후라이드 | D001 |

### 기술 스택
| 구분 | 기술 | 비용 |
|------|------|------|
| AI | OpenAI API | 사용량 기반 |
| 서버 | Python + FastAPI | 무료 |
| DB | SQLite → MySQL | 무료 |
| ERP 자동화 | pywinauto | 무료 |
| SMS 수집 | Tasker (안드로이드) | 약 4,500원 |
| 엑셀 처리 | pandas + openpyxl | 무료 |

### 개발 단계

**1단계: 데이터 준비 (가장 중요)**
- 최근 2~3개월 문자 발주 내용 수집 (최소 300건)
- 품목 마스터 엑셀 작성
- 거래처별 별칭 테이블 작성

**2단계: SMS 수집 자동화**
- 안드로이드 Tasker 설치 및 설정
- 문자 수신 시 서버로 자동 전송

**3단계: AI 주문 분석 모듈**
- OpenAI API 연동
- 품목코드 변환 로직 작성

**4단계: 주문 검수 화면**
- 간단한 웹 UI
- 담당자가 확인 후 승인

**5단계: ERP 자동 입력**
- pywinauto로 경영박사 화면 자동 조작
- 엑셀 저장

### 현재 진행 상태
- ✅ 전체 구조 설계 완료
- ✅ 기술 스택 선정 완료
- 🔴 데이터 수집 시작 안 됨 (첫 번째 할 일!)
- 🔴 실제 코드 작성 안 됨

## 프로젝트 3: 머니핏 (MoneyFit)

### 개요
소상공인을 위한 AI 경영관리 SaaS
- "매출은 아는데 왜 돈이 없는지 모르는" 사장님들의 문제를 해결
- 순이익, 원가율, 돈이 새는 곳을 한눈에 보여주는 프로그램

### 브랜드
- **공식명**: 머니핏 (MoneyFit)
- **애칭**: 머핏
- **포지셔닝**: 소상공인 AI 경영 비서

### 타겟 고객
- 1차 타겟: 아몬드치킨 35개 가맹점 (테스트 고객)
- 2차 타겟: 울산 지역 소상공인
- 3차 타겟: 전국 소상공인

### 핵심 차별화
기존 프로그램들과 다른 점:
1. **회계 지식 없어도 됨** - 사장님이 이해하는 언어로
2. **AI가 자동 분석** - "배달수수료가 왜 이렇게 많이 나왔는지" 직접 말해줌
3. **입력 최소화** - 영수증 촬영, PDF 업로드, POS 자동 연동
4. **오늘 마감 버튼** - 하루 요약 30초 완성

### MVP 기능 목록 (1차 개발)

**대시보드**
- 오늘 매출 / 순이익
- 이번 달 매출 / 순이익
- 경영 건강점수 (0~100점)
- 위험 신호 알림 🟢🟡🔴

**손익 분석**
- 원가율 34.8% (989,000원) ← 이 형식으로
- 인건비 18.5% (526,000원)
- 배달수수료 16.9% (480,000원)
- 광고비 2.3% (65,000원)

**오늘 마감 버튼**
- 버튼 하나로 하루 전체 요약
- 매출 / 순이익 / 원가율 / 특이사항

**AI 자동입력**
- 영수증 촬영 → AI 자동 분류
- 거래명세서 PDF → AI 분석
- 은행 문자 → 자동 수집

**재고관리**
- 입고 / 사용 / 폐기
- 재고 부족 자동 알림

**AI 질문 기능**
- "이번 달 왜 돈이 없지?" → AI가 분석해서 답변

### V2 기능 (POS 연동)
- OKPOS 연동 (1차)
- 포스뱅크 연동 (1차)
- KIS, 토스포스 (2차)

**배달앱 연동**

- 요기요

### 비즈니스 모델

- **확장**: 외부 소상공인으로 확대

### 현재 진행 상태
