## 1. 프로젝트 핵심 한 줄

**"왜 돈이 안 남지?"에 대한 자영업자의 걱정과 불안을 없애주는 AI 경영 동업자 앱**

- 브랜드명: 머니핏 (MoneyFit)
- 슬로건: 사장님은 장사에 집중하세요. 돈은 머니핏이 챙기겠습니다.
- 만든 사람: 이준호 대표 (아몬드치킨 프랜차이즈, 울산, 가맹점 38곳)

## 2. 절대 원칙 (모든 기능 결정의 기준)

이 질문을 통과 못하면 기능을 만들지 않는다:
**"이 기능이 사장님의 불안을 줄여주는가?"**

| 원칙 | 내용 |
|---|---|
| 미션 | "왜 돈이 안 남지?"의 불안을 없앤다 |
| 톤 | 동업자 — 친구도 비서도 아닌, 같이 고민하는 파트너 |
| 숫자 표시 | 항상 금액(원) 먼저, % 는 괄호 보조 |
| 평소 기준 | 최근 3개월 평균 |
| 상태 판정 | 🔥 +10% 이상 / ✅ ±10% / ⚠️ -10% 이하 |
| 대상 | 10대~70대 누구나, 3번 터치 안에 답 |

## 4. 완료된 화면 설계 상세

### 🏠 홈 화면

```
[상단바] 머니핏 로고 + 알림 벨 (시간 사용자 설정: 6시/7시/8시/9시/오픈 30분 전)

[동업자 한마디 카드] ← 가장 중요
  🔥 "이번달 진짜 잘 되고 있어요. 평소보다 +250만원 더 남을 것 같아요"
  ✅ "평소처럼 잘 가고 있어요. 걱정 안 하셔도 돼요"
  ⚠️ "이번달 살짝 늦어요. 같이 봐요 (-180만원)"

[월말 예측 카드]
  "이번달 이대로면 월말에..."
  금액 크게 → % 작게 옆에 → 평소 대비 금액 한 줄

[어제 한 줄] 매출 + 순이익 뱃지

[빠른 입력] [영수증 찍기(초록, 크게)] [직접 입력(흰색, 작게)]
```

**동업자 톤 원칙:**
- 🔥 같이 기뻐한다 ("진짜 잘 되고 있어요!")
- ✅ 안심시킨다 ("걱정 없어요")
- ⚠️ 혼자 두지 않는다 ("같이 봐요")

---

### 📸 영수증 AI 자동 입력 화면 (핵심 차별화)

**3단계 흐름:**
```
1단계 [촬영]
  → 카메라 뷰파인더 (노란 모서리 가이드)
  → "사진 찍기" 버튼

2단계 [AI 인식] (3초)
  → 항목별 로딩 바 애니메이션
  → 거래처 / 날짜 / 품목 / 수량 / 금액 / 부가세 순서로 인식

3단계 [확인 + 저장]
  → 인식 결과 표시 (항목별 수정 가능)
  → AI 한마디 (저장 전에 표시):
     "닭고기 단가 지난달보다 +8%. 이대로면 이번달 재료비 90만원 더 나갈 거예요"
  → "맞아요, 저장할게요" / "다시 찍을게요"

완료 후:
  → 누적 현황 업데이트 알림
  → "이번달 재료비 누적 1,240만원. 지난달 같은 시기보다 8% 높아요"
```

**AI 한마디 원칙:**
- 저장 직후 반드시 1줄
- 지난달 같은 품목 단가와 비교
- 이번달 영향 금액으로 환산 (추상적 % 금지)

---

### 📅 캘린더 화면

**구성:**
```
[상단] 월 이동 버튼 + 월 이름

[월 요약 바]
  매출 | 총 지출 | 순이익  (3분할, 어두운 초록 배경)

[캘린더 그리드]
  - 날짜마다 색상 점 하나:
    🟢 초록: 잘된 날 (순이익 +50만원 이상)
    🟡 노랑: 보통 (순이익 0~50만원)
    🔴 빨강: 적자
    ⚪ 회색: 기록 없음
  - 날짜 아래 순이익 금액 (9px, 작게)
  - 오늘 날짜: 초록 원으로 강조

[하단 상세 패널] (날짜 탭하면 열림)
  선택한 날짜 + 요일
  매출 / 지출 / 순이익 3칸 카드
  거래 내역 리스트 (아이콘 + 거래처명 + 항목 + 금액)
  기록 없는 날: "영수증 추가하기" 버튼
```

**색상 기준:**
- 순이익 +50만원 이상 → 초록
- 순이익 0~50만원 → 노랑
- 적자 → 빨강
- 기록 없음 → 회색

## 6. 현재 구현된 기능 (MoneyFit.html)

### 완료
- 매출/비용 입력 6개 카테고리
- 순이익 자동 계산
- AI 브리핑 (Claude Sonnet 4.6 API 실제 호출)
- 월별 추이 바 차트 (Recharts)
- 비용 구조 분석 + 자동 경고
- 세금 계산 (부가세 + 종합소득세 누진세율)
- 손익분기점 (BEP) 계산
- 데이터 저장 (localStorage)
- 아몬드치킨 6개월 샘플 데이터

### 아직 없음 (Claude Code에서 구현 순서)
1. 📅 캘린더 탭 → 날짜별 상세 (설계 완료, 구현 필요)
2. 🏠 홈 탭 재설계 → 동업자 한마디 + 월말 예측 (설계 완료, 구현 필요)
3. 📸 영수증 AI 입력 → Claude Vision API 연결
4. 📊 분석 탭 고도화
5. 🤖 AI 비서 탭 (대화형)
6. 알림 시간 설정 (사용자가 직접 지정)
7. PWA 전환 (폰 홈화면 설치)

## 7. 기술 스택

| 항목 | 내용 |
|---|---|
| 프레임워크 | React (CDN, Babel standalone) |
| 차트 | Recharts |

| 저장 | localStorage |
| 실행 방법 | MoneyFit.html 더블클릭 → 크롬에서 바로 열림 |
